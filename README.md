# RedisWebAdmin

由于经常操作Redis，所以业余时间写了一个简单的Web管理界面，方便自己和同事通过web页面操作Redis。

# 代码

基于PHP 5.5 + CI 2.2.6 + Bootstrap 3

# 演示站点

http://rediswebadmin.zuolinux.com
